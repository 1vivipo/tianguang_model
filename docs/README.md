# 天光AI - 自主训练系统

一个可以在手机浏览器中训练AI模型的系统。

## 功能特点

- 🧠 **模型训练** - 自定义参数训练语言模型
- 📊 **数据管理** - 上传、编辑训练数据
- 🧪 **测试评估** - 测试模型效果并评分
- 💬 **对话测试** - 与训练好的模型对话
- 💾 **模型导出** - 导出模型用于其他设备

## 使用方法

### 1. 添加训练数据

进入"数据管理"页面，输入训练数据（每行一条），或上传txt/json文件。

### 2. 配置训练参数

进入"模型训练"页面，设置：
- 训练轮次 (Epochs): 50-200
- 学习率 (Learning Rate): 0.001-0.1
- 批次大小 (Batch Size): 2-8（手机建议）

### 3. 开始训练

点击"开始训练"按钮，等待训练完成。

### 4. 测试模型

进入"测试评估"页面，输入提示文本测试生成效果。

### 5. 导出模型

进入"系统设置"页面，点击"导出模型"保存训练结果。

## 技术架构

- **前端**: 纯HTML/CSS/JavaScript
- **AI框架**: TensorFlow.js
- **加速**: WebGL/GPU

## 数据格式

### 文本格式 (txt)
```
人工智能是计算机科学的分支
深度学习是机器学习的重要组成部分
自然语言处理是AI的核心技术
```

### JSON格式
```json
[
  "人工智能是计算机科学的分支",
  "深度学习是机器学习的重要组成部分"
]
```

## 注意事项

1. 手机训练模型大小有限，建议数据量在100-1000条
2. 训练时建议连接充电器
3. 定期保存模型避免丢失
