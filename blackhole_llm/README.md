## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python -m api.server
```

### 3. 观察成长

```
部署后自动运行：

每5分钟：
  - 100只虫子同时出击
  - 爬取100个领域知识
  - 吞噬进入知识黑洞

每10分钟：
  - 10条神经同时训练
  - 并行学习新知识
  - 模型持续优化

每天：
  - 爬取约 28,800 条知识
  - 训练约 1,440 次
  - 知识量爆炸增长
```

### 4. 调用API

```bash
curl -X POST http://localhost:8080/api/chat \
     -H "X-API-KEY: sk-blackhole-2024-swarm" \
     -d '{"message": "最近科技新闻"}'
```

## 📡 API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 服务信息 |
| `/api/chat` | POST | 对话 |
| `/api/status` | GET | 状态监控 |
| `/api/crawl` | POST | 手动触发爬取 |
| `/api/train` | POST | 手动触发训练 |
| `/api/knowledge` | GET | 知识列表 |
| `/api/domains` | GET | 领域统计 |

## ⚙️ 配置

```python
from config import Config

config = Config()

# 虫群配置
config.swarm.num_crawlers = 100    # 100只虫子
config.swarm.crawl_interval = 300  # 5分钟爬一次

# 神经网络配置
config.neural.num_networks = 10    # 10条神经
config.neural.train_interval = 600 # 10分钟学一次
```
