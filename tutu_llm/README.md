## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python -m api.server
```

### 3. 等待学习

突突会自动：
- 每5分钟爬取新闻和知识
- 每10分钟训练学习
- 自动分类整理知识库

### 4. 调用API

```bash
curl -X POST http://localhost:8080/api/chat \
     -H "X-API-KEY: sk-tutu-2024-auto-learn" \
     -d '{"message": "最近有什么新闻"}'
```

## 📡 API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 服务信息 |
| `/api/chat` | POST | 对话 |
| `/api/status` | GET | 状态监控 |
| `/api/crawl` | POST | 手动触发爬取 |
| `/api/learn` | POST | 手动触发学习 |
| `/api/knowledge` | GET | 知识列表 |

## ⚙️ 配置

```python
from config import TutuConfig

config = TutuConfig()

# 爬虫配置
config.crawler.crawl_interval = 300  # 5分钟爬一次
config.crawler.keywords = ["AI", "科技", "编程"]

# 学习配置
config.learner.learn_interval = 600  # 10分钟学一次
config.learner.batch_size = 32
```

## 💡 工作原理

### 自动爬取
```
1. 定时访问新闻网站
2. 提取正文内容
3. 过滤广告垃圾
4. 存入知识库
```

### 自动分类
```
1. 分析内容关键词
2. 匹配分类规则
3. 提取关键信息
4. 生成摘要
```

### 自动学习
```
1. 从知识库采样
2. 构建训练数据
3. 更新模型权重
4. 保存检查点
```
